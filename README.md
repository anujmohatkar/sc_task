# README

### Production URL
    https://sc-task.onrender.com

### JSON API USAGE

* send a get request to `/api/max_sum` and send a your numbers as json body
    ```json
    {
        "numbers": [4,2,1,-3,2,0]
    }
    ```

response
    ```json
    {
        "max_sum": [
            4,
            2,
            1
        ]
    }
    ```
